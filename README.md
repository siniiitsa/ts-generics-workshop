# TS Generics Workshop

Воркшоп про generics у TypeScript

## Структура

- `demo/` — готові приклади для пояснення теми. Читайте файли по порядку: від `1` до `6`.
- `problems/1-basic`, `problems/2-interfaces`, `problems/3-constraints`, `problems/4-keyof` — основні практичні блоки по 3 задачі.

## Setup

1. Встановіть Node.js 18 або новіше.
2. Виконайте `npm install`.
3. Перевірте матеріали командою `npm run typecheck`.

## Як працювати з матеріалами

- Файли в `demo/` уже вирівняні під пояснення і можуть використовуватись як готові приклади.
- Файли в `problems/` навмисно стартують з `any`, щоб учасники замінили їх на generics під час практики.
- Рядки з коментарем `має бути помилка TypeScript` слугують чекпойнтами для готового розв'язку.
